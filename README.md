# 🍲 AI Hotpot

Open-source AI skills and recipes — ready to remix.

Each folder is a self-contained **skill**: a set of instructions, templates, and scripts that an AI agent (or a human) can follow to accomplish something interesting.

## Skills

| Skill | Description |
|-------|-------------|
| [ai-lesson-for-kids](./ai-lesson-for-kids/) | Run a 45-minute interactive AI lesson for kids age 7-10 with a live website, Teachable Machine, and AI-generated art |

## What's a Skill?

A skill is a portable, reusable recipe for AI-assisted work. Each one includes:

- **SKILL.md** — Instructions an AI agent can follow (or a human can read)
- **assets/** — Templates, HTML files, images
- **scripts/** — Automation scripts (bash, python, etc.)
- **references/** — Background material, lesson plans, research

Skills are designed for [OpenClaw](https://github.com/openclaw/openclaw) but work with any AI agent platform — or no platform at all. The instructions are human-readable; the scripts are standalone.

## Contributing

Got a skill to share? Open a PR! Each skill should be self-contained in its own folder with a SKILL.md at the root.
